Introduction                    1.1 Purpose              The purpose of this Software Requirements Specification (SRS) document is to provide a complete, unambiguous, and verifiable description of the requirements for the Staff Sync – Integrated Workplace Platform developed for BIG GROW LANKA (PVT) LTD. This document serves as the contractual baseline between the developer (A.M.C.D Adhikari) and the client organisation. It guides the design, implementation, testing, and deployment phases of the project. All functional and non-functional requirements stated herein are subject to review and approval by the project supervisor and the client.                                1.2 Intended Audience              This document is intended for the following audience:                                                               Audience                          Purpose                              Developer (A.M.C.D Adhikari)                   Implement the system as per specified requirements                         Supervisor (Mr. K.W.G.S. Konthasinghe)                   Evaluate and guide the project development                         Client (BIG GROW LANKA Manager)                   Validate requirements and accept deliverables                         Future Maintainers                   Understand system design for future enhancements                                   1.3 Product Scope              Staff Sync is a cross-platform mobile application (Flutter + Firebase) and a web-based admin panel designed exclusively for the internal use of BIG GROW LANKA (PVT) LTD, an import and trading company in Sri Lanka. The system addresses the company's need for a unified, secure, and efficient digital platform by replacing fragmented communication methods (Gmail, WhatsApp, phone calls, physical attendance books) with a single integrated solution.                          The system provides the following core capabilities:                                            Role-based internal chat (1-1 and group) with document and image sharing                                            Announcement management with role-based visibility (Owner: global; Department Head: department-only) and view tracking                                            User management (CRUD) via an Admin/Manager dashboard (web-based)                                            Strong authentication: email/password with optional biometric login (face/fingerprint)                                            QR code-based digital attendance with clock-in/clock-out and report export                                            Emergency SOS live location sharing for field employees                                            Leave request management – employees apply for leave; admin approves or rejects                                            Real-time push notifications for messages, announcements, attendance, and leave status updates                          Out of scope for Phase 1: External payroll system integration, advanced analytics dashboard with charts, iOS App Store deployment, automatic leave calculation/carry-forward, and dynamic attendance filtering by department.                                                                                                                                                                                                                                                                                                                                   2. General Description                    2.1 Product Perspective              Staff Sync is a self-contained mobile and web-based application that operates independently of any external HR or ERP systems in its first phase. The system is developed from scratch to replace the current ad-hoc communication setup (Gmail, WhatsApp, physical records) used by BIG GROW LANKA.
